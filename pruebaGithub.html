Plantamiento de un programa de compra online

Para un programa basico de compra online de cara a un empleado de nuestra tienda necesitamos:

  1. Un metodo que se encargue de mostrar un menu con las opciones que puede
    efectuar el empleado por medio de un switch.

  2. Un bucle while que repita el metodo principal del menu una y otra vez hasta que el usuario elija la
    opcion salir, opcion que rompera el bucle para que el programa termine.

  3. Necesitamos uno o varios metodos donde tengamos los Strings o sysos que se mostraran un texto,
    junto a cada una de las opciones de nuestro menu, indicando para que sirve cada opcion disponible.
    Nuestro menu se encargara de llamar a este metodo mas adelante.

  4. Un metodo con un scanner y un teclado para que el usuario pueda seleccionar
    una opcion de las disponibles en nuestro menu.

  5. Objetos que se encarguen de representar los productos disponibles en nuestra
    tienda (ej:Objeto producto y sus hijos ---> objeto Libro, objeto tablet, objeto auriculares etc.).

  6. Una clase que posea una lista dinamica como un arrayList, por ejemplo, que se encargue de
    almacenar,introducir,modificar y eliminar los productos de nuestra tienda.

El menu de nuestra tienda se mostraria de la siguiente manera:

[Se ejecuta el metodo menu,se crea una variable que llama y almacena al metodo Scanner
para que el usuario pueda seleccionar las opciones disponibles en el menu,
menu llama al metodo mostrar texto]
[Se muestra el siguiente texto]

Bienvenido al gestor de SevillaExpress

[Se inicia el bucle while que se repetira mientras que la opcion seleccionada por el usuario
sea menor que 5].

Seleccione una de las opciones disponibles:

  1. Añadir un objeto a la tienda
  2. Eliminar un objeto de la tienda
  3. Modificar un objeto de la tienda
  4. Añadir o quitar stock de un objeto en tienda
  5. Salir
  • Si el usuario selecciona la opcion 1, se mostrara otro texto: "Introduzca nombre del producto, precio y stock disponible". Una vez introducido los datos se creara un objeto con dicha informacion que pasara a almacenarse en el ArrayList.

  • Si el usuario selecciona la opcion 2, se mostrara el siguiente texto: "Introduzca el nombre del producto que desea eliminar". Una vez introducido el nombre se buscara dentro del ArrayList un producto existene con dicho nombre,
    si el producto existe, se borrara del ArrayList y se eliminara dicho objeto.

  • Si el usuario elije la opcion 3 se mostrara el siguiente texto: "Introduzca el nombre del producto que desea
    modificar". El ArrayList hara una busqueda para encontrar un producto con un nombre que coincida con el
    nombre introducido con el usuario, si dicho producto existe se muestra el siguiente mensaje:

    "¿Que desea modificar?"

    1. Nombre del producto
    2. Precio del producto

    Idependientemente de las opciones que el usuario escoja, los nuevos datos que introduzca se modificaran por medio de los setters del objeto.

  • Si el usuario selecciona opcion 4,el programa le preguntara por el nombre del objeto, una vez introducido, el array
    buscara dicho objeto por su nombre, si este objeto existe, el programa indicara al usuario dos opciones seleccionables

    1. Añadir stock
    2. Retirar stock

    Si el usuario elige la opcion 1 e introduce 8 para aumentar el stock de unidades de una tablet, estas 8 se sumaran a las unidades ya existentes previamente de tablets,para evitar que entren numeros negativos se comprobara que el
    numero introducido sea mayor a 0 con un condicional(ej: if(numero>0{this.stock+=numero})). Por el contrario si-
    elige la opcion 2, esas 8 unidades se restarian no sin antes comprobar por medio de un condicional que el numero de unidades disponibles, es mayor o igual al numero de unidades de stock que se van a restar (ej: if(this.stock>=numero{this.stock-=numero; })), para evitar bugs en el programa

  • Si el usuario elige la opcion 5 se rompera el bucle y acabara el programa
